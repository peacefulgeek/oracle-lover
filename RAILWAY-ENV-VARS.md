# Railway Environment Variables

Set these in your Railway dashboard under **Variables**:

| Variable | Description |
|----------|-------------|
| `ANTHROPIC_API_KEY` | Claude API key for article generation |
| `FAL_API_KEY` | FAL.ai key for hero image generation |
| `GH_PAT` | GitHub PAT for auto-push to peacefulgeek/oracle-lover |
| `BUNNY_STORAGE_KEY` | Bunny CDN storage API key (oracle-lover zone) |
| `AUTO_GEN_ENABLED` | Set to `true` to enable weekly article generation |
| `PORT` | Server port (Railway sets automatically, default 3000) |
